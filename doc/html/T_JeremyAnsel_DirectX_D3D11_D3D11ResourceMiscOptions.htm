  Member nameValueDescription
None0 No option.
GenerateMips1 Enables MIP map generation by using GenerateMips(D3D11ShaderResourceView) on a texture resource. The resource must be created with the bind flags that specify that the resource is a render target and a shader resource.
Shared2 Enables resource data sharing between two or more Direct3D devices. The only resources that can be shared are 2D non-mipmapped textures.
TextureCube4 Sets a resource to be a cube texture created from a Texture2DArray that contains 6 textures.
DrawIndirectArgs16 Enables instancing of GPU-generated content.
BufferAllowRawViews32 Enables a resource as a byte address buffer.
BufferStructured64 Enables a resource as a structured buffer.
ResourceClamp128 Enables a resource with MIP map clamping for use with SetResourceMinLod(D3D11Resource, Single).
SharedKeyedMutex256 Enables the resource to be synchronized by using the AcquireSync(UInt64, UInt32) and ReleaseSync(UInt64) APIs.
GdiCompatible512 Enables a resource compatible with GDI.
SharedNTHandle2048 Set this flag to enable the use of NT HANDLE values when you create a shared resource. By enabling this flag, you deprecate the use of existing HANDLE values.
RestrictedContent4096 Set this flag to indicate that the resource might contain protected content; therefore, the operating system should use the resource only when the driver and hardware support content protection. If the driver and hardware do not support content protection and you try to create a resource with this flag, the resource creation fails.
RestrictSharedResource8192 Set this flag to indicate that the operating system restricts access to the shared surface.
RestrictSharedResourceDriver16384 Set this flag to indicate that the driver restricts access to the shared surface.
Guarded32768 Set this flag to indicate that the resource is guarded.
TilePool131072 Set this flag to indicate that the resource is a tile pool.
Tiled262144 Set this flag to indicate that the resource is a tiled resource.
