points
Type: JeremyAnsel.DirectX.D2D1D2D1Point2F
An array of one or more points that describe the lines to draw. A line is drawn from the geometry sink's current point (the end point of the last segment drawn or the location specified by BeginFigure(D2D1Point2F, D2D1FigureBegin)) to the first point in the array. if the array contains additional points, a line is drawn from the first point to the second point in the array, from the second point to the third point, and so on.
